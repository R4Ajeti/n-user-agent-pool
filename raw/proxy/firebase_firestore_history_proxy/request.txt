Endpoint:
https://firestore.googleapis.com/v1/projects/${FIREBASE_FIRESTORE_PROJECT_ID}/databases/(default)/documents/userAgentHistory

Method:
POST

Headers:
Accept: application/json
Content-Type: application/json
User-Agent: n-user-agent-pool/1.0.0
Authorization: Bearer <decoded-token-from-env>

Environment:
FIREBASE_FIRESTORE_CREDENTIAL_BASE64 contains base64-encoded JSON with an
in-memory token field such as accessToken, authToken, idToken, or token.
FIREBASE_FIRESTORE_PROJECT_ID contains the Firebase project ID.

Purpose:
Append a safe user-agent history record to Cloud Firestore when
USER_AGENT_HISTORY_BACKEND=firebase_firestore.

Safety:
The decoded Firebase credential is never logged, written to disk, or included in
raw examples. This file intentionally uses placeholders only.
